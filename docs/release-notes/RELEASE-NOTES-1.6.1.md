# ICL 1.6.1 Release Notes

## New Features

### ocicl Integration
- `,libyear` - Show dependency freshness metric (how out-of-date your dependencies are)
- `,changes [system]` - Show LLM-generated changelogs for packages
- Tab completion for `,changes` uses ocicl-runtime to list available systems

## Improvements

- Normalized ocicl naming (lowercase) throughout codebase and documentation

## Breaking Changes

None.
